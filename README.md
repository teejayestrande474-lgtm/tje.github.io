# CICS SJCBI Website

A static informational website for the **College of Information and Computing Sciences (CICS)** at **St. Joseph College of Baggao, Inc. (SJCBI)**.
It provides prospective students, faculty, and visitors with essential information such as academic programs, institutional vision and mission, faculty profiles, news, and contact details.
The site uses a clean, responsive layout with embedded CSS.

---

## Features

* **Responsive Design**
  Adapts smoothly to different screen sizes using flexbox and media queries.

* **Simple Navigation**
  A structured navigation bar links users to key sections.

* **Content Sections**

  * **Home:** Welcome message and brief overview
  * **Programs:** Information about BSIT and BLIS offerings
  * **Vision & Mission:** Institutional purpose and goals
  * **Faculty:** Profiles and photos of core faculty members
  * **News & Events:** Links to Facebook posts and department updates
  * **Contact:** Footer with address and communication details

* **Visual Elements**
  Includes official logos, faculty photos, and a color palette aligned with SJCBI branding (green and yellow).

* **Accessibility**
  Uses semantic HTML and descriptive alt text for images.

---

## Technologies Used

* **HTML5** for structure and content
* **CSS3** for styling, layout, and responsiveness (embedded `<style>` block)

---

## How to View

1. Download or clone the repository.
2. Open **index.html** in any modern browser (Chrome, Firefox, Edge, Safari).
3. No server setup is required—this is a fully static site.

---

## File Structure

* **index.html** – Main file containing all content and embedded CSS.

---

## Customization

* **Images:** Update the `src` attributes in `<img>` tags. Placeholder fallbacks are included via `onerror`.
* **Text Content:** Modify headings and paragraphs to update information.
* **Styling:** Edit the `<style>` section to adjust colors, fonts, or layout rules.
* **Links:** Replace placeholder `href="#"` values with actual URLs.

---

## Browser Compatibility

* Works on all modern browsers supporting HTML5 and CSS3.
* Older browsers (e.g., IE11) may not be fully supported.
